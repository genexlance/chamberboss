Plugin Documentation

Installation

  1. Download the plugin ZIP file from the main page
  2. Go to your WordPress admin dashboard
  3. Navigate to Plugins > Add New
  4. Click "Upload Plugin" at the top
  5. Choose the downloaded ZIP file
  6. Click "Install Now"
  7. After installation, click "Activate Plugin"

Configuration

Stripe Settings

After activating the plugin, you'll need to configure your Stripe API keys:

  1. Go to Chamber Members > Settings in your WordPress admin
  2. Enter your Stripe Publishable Key and Secret Key
  3. If you're testing, enable Test Mode
  4. Save the settings

MailPoet Integration

To enable MailPoet integration:

  1. Make sure the MailPoet plugin is installed and activated
  2. In the Chamber Membership Manager settings, configure your MailPoet lists
  3. Save the settings

Using the Plugin

Business Directory

To display the business directory on your site:

  1. Create a new page or edit an existing one
  2. Add the shortcode [chamber-directory] to the page content
  3. Publish or update the page

Member Registration

Members can register by:

  1. Visiting your site's registration page
  2. Filling out the registration form
  3. Processing payment through Stripe
  4. Creating their business listing after successful payment

Troubleshooting

Common Issues

  • Plugin not activating: Make sure you're using WordPress 6.0 or higher and PHP 8.0 or higher
  • Stripe payments not working: Check that your API keys are correct and that you're using the right mode (test/live)
  • Directory not showing listings: Make sure listings are published and not pending approval

Getting Support

If you're experiencing issues not covered in this documentation, please contact our support team at support@chamber-plugins.com
